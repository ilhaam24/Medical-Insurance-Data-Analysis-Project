# Medical-Insurance-Data-Analysis-Project
This Data Analysis Project aims to clean a Medical Insurance Dataset collected from Kaggle, clean and import the clean dataset by Python, and then create a Dashboard with PowerBI. The Dashboard aims to answer some significant Business problems and recommend some solutions.


#Step-1
At the start of this project, I collected a dataset from Kaggle for my data analysis project. The raw dataset has been attached with my repository, which is "insurance.csv". The raw dataset had duplicate rows and other inconsistencies that may harm the data analysis.

#Step-2
So, for that I cleaned the dataset with python. I removed the duplicate rows and added an additional column, which is "BMI category". This column categorizes individuals in three weight categories, Normal, Overweight, and Obese, based on their BMI. Then I imported the cleaned dataset into a CSV file, which is "Cleaned_insurance_data.csv".

#Step-3
For data visualization, I used powerBI. I used 4 types of visualization for data analysis on this project, which are scatter plot, clustered bar chart, line chart, and decomposition tree.

#Step-4
With the data visualizations, I mainly wanted to answer 4 significant business questions, which are 
1. What is the single biggest driver of medical costs, and how should it impact our premiums?
2. What happens when high BMI compounds with other risk factors?
3. Which region is bleeding the most money, and why?
4. How does family size impact cost predictability?
